Legacy JSON State Demo
======================

This folder archives the first scaffold of the project, which used a JSON file
as the only persistent state store.

The current MVP uses SQLite through:

- `src/storage/database.py`
- `src/tools/sqlite_tools.py`
- `src/agent/workflow.py`
- `app.py`

Archived files here are kept only as historical reference:

- `data/initial_state.json`
- `src/game/state.py`
- `src/tools/game_tools.py`
- `scripts/demo_tools.py`
- `scripts/inspect_state.py`
