# AI Usage Statement

本项目允许合理使用 AI 工具辅助开发，但需要在课程报告中如实说明。

## 自己完成或需要自己确认的部分

- 项目选题和课程要求匹配；
- 系统目标与 MVP 范围确认；
- Agent workflow 的设计取舍；
- 演示案例是否符合课程展示目标；
- 本地运行、截图、录屏和结果核验；
- 报告中的分析、反思和结论。

## AI 辅助的部分

- 部分 Python 代码模板；
- Streamlit 页面结构草稿；
- SQLite schema 草稿；
- README 和文档初稿；
- 测试用例草稿；
- 调试建议和错误定位。

## 不应做的事情

- 不应伪造运行结果；
- 不应把未运行过的效果写成已完成；
- 不应把 AI 生成的代码直接视为个人全部工作；
- 不应只提交聊天记录或平台配置，而没有可运行成品。

## 建议写入报告的表述

```text
本项目在实现过程中使用 AI 工具辅助生成部分代码模板、文档草稿和调试建议。项目的选题定位、系统架构确认、模块整合、运行验证、实验案例设计和结果分析由项目成员完成。报告中的运行截图和日志均来自本地真实运行结果。
```
